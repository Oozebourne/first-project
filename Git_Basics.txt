git init - инициализация репозитория
git status - статус репозитория
git add - добавить файл в репозиторий
git add --all либо git add . - добавить все файлы в папке в репозиторий
git commit -m 'Message' - запомнить изменения в файлах репозитория, добавить в историю изменений с сообщением Message.
git remote add name SSH_URL - привязать к удалённому репозиторию SSH_URL локальный репозиторий name.
git remote -v - подробная информация о привязанном репозитории.
git push - отправить изменения на привязанный удалённый репозиторий.
